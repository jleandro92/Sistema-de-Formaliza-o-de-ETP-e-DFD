⚙️ Requisitos Não Funcionais (RNF)

👉 Como o sistema deve se comportar

RNF01 – O sistema deve possuir interface web responsiva.

RNF02 – O sistema deve ser acessível por navegadores modernos.

RNF03 – O sistema deve garantir autenticação segura.

RNF04 – O sistema deve manter a integridade dos dados armazenados.

RNF05 – O sistema deve apresentar tempo de resposta adequado.

RNF06 – O sistema deve permitir escalabilidade para novos módulos.

RNF07 – O sistema deve utilizar banco de dados relacional.

RNF08 – O sistema deve permitir backup dos dados.

RNF09 – O sistema deve ser desenvolvido com tecnologias amplamente utilizadas no mercado.

RNF10 – O sistema deve seguir boas práticas de desenvolvimento de software.
